Check if N numbers is sorted ascending Open on Hyperskill

Write a program that checks if N numbers are sorted ascending (from the smallest number to the largest one).


The first line contains the number N.
The second line contains N numbers separated by spaces.

Output "YES" if N numbers are sorted ascending, otherwise, output "NO".


Sample Input:
4
1 2 3 4

Sample Output:
YES

Sample Input:
4
1 2 3 0

Sample Output:
NO


Memory limit: 256 Mb
Time limit: 5s
